Forms

Forms are a collection of input fields within HTML. The user can enter different types of information depending on the type of input field. When the form is submitted, the browser will collect all the inputs and either send them to a url (if the attribute action is present on the form), or will send them to a JavaScript function (if the attribute onsubmit is present).
